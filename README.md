GERENCIAMENTO DE ESTOQUE EM PYTHON

O Gerenciamento de Estoque é um projeto desenvolvido em Python com o objetivo de controlar produtos de forma simples e eficiente através do terminal. O sistema permite cadastrar, listar, remover e atualizar produtos, além de armazenar todas as informações em um arquivo JSON, garantindo a persistência dos dados mesmo após o encerramento da aplicação.

Objetivo do Projeto

Este projeto foi desenvolvido para praticar conceitos fundamentais da programação em Python, aplicando conhecimentos de lógica de programação e manipulação de dados.

Durante o desenvolvimento foram utilizados conceitos como:

* Funções;
* Estruturas condicionais;
* Laços de repetição;
* Manipulação de arquivos JSON;
* Dicionários e estruturas de dados;
* Validação de entradas do usuário;
* Organização de código;
* Persistência de dados.

Funcionalidades

* Cadastro de produtos;
* Armazenamento de nome, marca e quantidade;
* Geração automática de ID para cada produto;
* Listagem completa dos produtos cadastrados;
* Remoção de produtos pelo ID;
* Adição de estoque;
* Remoção de estoque;
* Controle de quantidade disponível;
* Alerta de produtos com estoque baixo;
* Salvamento automático dos dados em arquivo JSON;
* Menu interativo via terminal.

Tecnologias Utilizadas

* Python
* Biblioteca JSON
* Terminal / Console
* Lógica de Programação

Sobre o Projeto

O Gerenciamento de Estoque foi criado com foco no aprendizado e na aplicação prática dos conceitos de Python. O sistema simula operações básicas encontradas em softwares de controle de estoque utilizados por empresas e comércios.

Além de fortalecer conhecimentos em programação, o projeto permitiu trabalhar com armazenamento de dados em arquivos JSON, organização de funções e desenvolvimento de aplicações interativas executadas diretamente no terminal.
